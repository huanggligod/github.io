ES6 Class

javascript生成实例的传统方法是通过构造函数

传统方法
JS原型与继承

ES6直接引用Class（类）概念，通过class关键字来定义类,一个类必须有constructor方法，没有显示定义，会默认添加一个空的constructor方法，constructor方法默认返回实例对象（this),如果constructor返回一个新的对象，会导致实例对象不是类的实例；类必须使用new调用，

ES6中，生成类的实例对象的写法与ES5完全一样，也是使用new命令。实例的属性除非定义在其本身（this)上，否则都在类上（class)

class的继承

class 可以通过extends 关键字实现继承,可继承类中属性和方法(static定义的静态方法不会被实例继承),ES5的继承，实质是先创造子类的this对象，然后将父类的属性和方法添加到子类的this对象
（parent.apply(this)）,ES6是先创造父类this对象，子类必须先调用super方法

子类必须在构造函数constructor方法中调用super方法，因为子类没有自己的this,super方法会使子类继承父类的this对象，子类如果没有constructor方法会被默认添加，super虽然代表父类的构造函数，返回的子类的this

Object.getPrototypeOf()方法可以用来判断一个类是否继承另一个类
